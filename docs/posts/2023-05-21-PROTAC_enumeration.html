PROTAC

PROTAC is a class of small-molecule compounds designed to target and degrade specific proteins within cells. It is a strategy used in drug development to selectively remove disease-causing proteins by utilizing the cell’s own protein degradation machinery.

The typical structure of a PROTAC molecule consists of three main components: a ligand for the target protein (called the “targeting moiety”), a ligand for an E3 ubiquitin ligase enzyme (called the “recruiting moiety”), and a linker connecting the two moieties.

The idea behind PROTACs is that they can bind to both the target protein and the E3 ubiquitin ligase simultaneously. This interaction leads to the transfer of ubiquitin molecules from the E3 ligase to the target protein, resulting in the labeling of the target protein for degradation by the proteasome. As a result, the target protein is degraded, leading to its functional inactivation.

PROTACs have gained significant attention in the field of drug discovery and have shown promise in targeting various disease-causing proteins, including cancer-related proteins and other therapeutic targets. They offer a potential alternative to traditional small-molecule inhibitors, which often only block the activity of the target protein without leading to its degradation.

For more information on PROTAC can be obtained thorugh fiew review papers [1],[2]

The post describes how the rdkit can be used for create PROTAC with different linker length and also with different linker compoistion between the target and ligase ligand.

# Functions for generating the linker library

def get_atom_index(mol):
    # Get linking position atom index
    link_atom_index = []
    for atom in mol.GetAtoms():
        if atom.GetSymbol() == '*':
            link_atom_index.append(atom.GetIdx())
    return link_atom_index

def delete_last_atom(mol,del_atoms=1):
    first_mol = Chem.RWMol(mol)
    input_idx = [atom.GetIdx() for atom in first_mol.GetAtoms()]
    link_mol = Chem.EditableMol(mol)
    link_mol.RemoveAtom(input_idx[-1]) #Remove last atom 
    first_mol = link_mol.GetMol()
    return first_mol

def generate_polymer(input_mol,num=1,del_atoms=1):
    # Frag for polymer
    frag_mol = Chem.RWMol(input_mol)
    input_idx = [atom.GetIdx() for atom in frag_mol.GetAtoms()]
    for deln in range(del_atoms):
        link_mol = Chem.EditableMol(frag_mol)
        link_mol.RemoveAtom(input_idx[0]) # Remove first atom for the fragment polymer
        frag_mol = link_mol.GetMol()
    linker = ''
    for inum in range(num):
        if not linker:
            linker = Chem.RWMol(input_mol)
        else:
            linker = delete_last_atom(linker,del_atoms=del_atoms)
            linker = Chem.CombineMols(linker,frag_mol)
            link_idx = []
            for atom in linker.GetAtoms():
                if (atom.GetAtomicNum() != 0) and (len(atom.GetNeighbors()) == 1):
                    link_idx.append(atom.GetIdx())
            link_mol = Chem.EditableMol(linker)
            link_mol.AddBond(link_idx[0],link_idx[1],order=Chem.rdchem.BondType.SINGLE)
            linker = link_mol.GetMol()  
    return linker
def combine_frags(front_mol,middle_mol):
    Front_idx = [ atom.GetIdx() for atom in front_mol.GetAtoms() if (atom.GetAtomicNum() == 0)]
    if len(Front_idx) == 1:
        F_index = Front_idx[0]
    else:
        F_index = Front_idx[1]
    Front_neighbor = front_mol.GetAtomWithIdx(F_index).GetNeighbors()[0].GetIdx()
    Middle_idx = [ atom.GetIdx() for atom in middle_mol.GetAtoms() if (atom.GetAtomicNum() == 0)]
    Middle_neighbor = middle_mol.GetAtomWithIdx(Middle_idx[0]).GetNeighbors()[0].GetIdx()
    merge_mol = Chem.CombineMols(front_mol,middle_mol)
    link_mol = Chem.EditableMol(merge_mol)
    link_mol.AddBond(Front_neighbor,Middle_neighbor+front_mol.GetNumAtoms(),order=Chem.rdchem.BondType.SINGLE)
    link_mol.RemoveAtom(Middle_idx[0]+front_mol.GetNumAtoms()) # Delete larger index number first
    link_mol.RemoveAtom(F_index)
    merge_mol = link_mol.GetMol()
    return merge_mol

Generate methyl linker

Here, we will generate different length of methyl linkers which connects between the target and ligase ligand.

def generate_methyl_linker(num=1):
    methyl = Chem.MolFromSmiles('*-C-*')
    linker_lib = []
    for idx in range(1,num+1):
        linker = generate_polymer(methyl,num=idx,del_atoms=1)
        linker_lib.append(linker)
    return linker_lib

def generate_peg_linker(num=1):
    linker_lib = []
    peg = Chem.MolFromSmiles('*-OCCO-*')
    for idx in range(1,num+1):
        linker = generate_polymer(peg,num=idx,del_atoms=2)
        linker_lib.append(linker)
    return linker_lib

Custom linker composition

Case-1

Here we will create linker library which composed of three parts: part-1(methyl), part-2 (piperidine) and part-3 (peg).

The methyl and peg number can be varied from 1 to n. Here, we will try to generate n=2. The proposed PROTAC library will be like

target - (methyl)1 - piperidine - (peg)1 - ligase

target - (methyl)2 - piperidine - (peg)2 - ligase

methyl_part = generate_methyl_linker(num=2) # Part-1
peg_part = generate_peg_linker(num=2) # Part-3
piperidine = Chem.MolFromSmiles('*-N1CCN(-*)CC1 ') # Part-2
PROTAC_lib = []
for i in range(2):
    for Li in LIGASE:
        linker = combine_frags(methyl_part[i],piperidine) # Combine part-1 and part-2
        linker = combine_frags(linker,peg_part[i]) # Combine part-3
        protac_temp = combine_frags(target,linker) # Combine target ligand and linker
        protac_temp = combine_frags(protac_temp,Li) # Combine target-linker to ligase ligand
        PROTAC_lib.append([Chem.MolToSmiles(target),Chem.MolToSmiles(Li),Chem.MolToSmiles(linker),Chem.MolToSmiles(protac_temp)])
PROTAC_DICT = pd.DataFrame(columns=['target','ligase','linker','protac'])
for i in PROTAC_lib:
    TEMP_DICT = pd.DataFrame(dict(zip(['target','ligase','linker','protac'],i)),index=[0])
    PROTAC_DICT = pd.concat([PROTAC_DICT,TEMP_DICT], ignore_index=True)
